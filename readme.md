## 🛠 Tech Stack

### 🎨 Frontend

- ⚛️ **React**
- 🎨 **TailwindCSS** + CSS
- 🔑 **Clerk Auth**
- 🌐 HTML

### 🖥 Backend

- 🌍 **Node.js** + Express.js
- 🛢 **Prisma ORM**
- 🗄 **MySQL** (Cloud Database)
- ☁️ **Cloudinary** (Image Storage)

## 🚀 Getting Started

### 🔧 Prerequisites

- Node.js >= 18
- MySQL Database (local or cloud)
- Clerk API Keys
- Cloudinary Account

### ⚙️ Installation

```bash
# Clone the repository
git clone https://github.com/your-username/your-repo.git
cd your-repo

# Install dependencies
npm install

# Setup environment variables
cp .env.example .env
```
